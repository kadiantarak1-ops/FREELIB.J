# 📚 FreeLib

FreeLib is a lightweight, web-based online library that allows users to explore and read books directly from their browser. Designed with simplicity and accessibility in mind, the platform provides a clean and modern interface where users can easily search for books and discover new reads across multiple genres.

The website features a smart search system that helps users quickly find books by title or author. Results are ranked based on relevance, ensuring that users get the closest matches to their queries. The general library offers a unique reading experience alongside classic literature.

All books available on the platform are sourced from public domain libraries, making them free and legally accessible. Users can open and read books within the website itself using an embedded viewer, creating a smooth and uninterrupted reading experience without needing to download files or visit external pages.

FreeLib is built entirely using HTML, CSS, and vanilla JavaScript, making it fast, responsive, and easy to deploy. It requires no setup or installation—simply open the file in a browser to start using it.

This project is ideal for beginners exploring web development, as well as for readers looking for a minimal and distraction-free digital library experience.
